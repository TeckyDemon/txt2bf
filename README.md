## Prerequisites

### Windows

Install **make** and **mingw**.

## Installation

```
git clone "https://github.com/DeBos99/txt2bf.git"
make -C txt2bf
```

## Usage

`./txt2bf INPUT_FILE OUTPUT_FILE`
